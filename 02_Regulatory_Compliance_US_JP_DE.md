# 02_Regulatory_Compliance — 検品後・出荷前の法令適合レイヤー

本ドキュメントは、**力学スクリーニング**および **Cross Industry Screening** の**両方が完了した後**に参照する。検品完了後の最終ゲートとして、ハードウェア出荷時に満たすべき法令・認証条件を US・Japan・Germany（EU）の 3 文化圏で整理する。

---

## 参照条件：スクリーニング完了後

本ドキュメントは以下の**実行順序**の 3 番目として参照する。

```
[1. 力学スクリーニング] → [2. Cross Industry Screening] → [3. 本ドキュメント: 法令・国際規格の参照]
```

- **力学スクリーニング**（`00_Screening.md` § 適用ドメイン）で適用物理ドメインを確定する。
- **Cross Industry Screening**（`00_Screening.md` § Cross Industry Screening）で産業種別を確定する。
- 上記完了後、本ドキュメントおよび産業別の追加法令・国際規格を参照する。

---

## 位置づけ：検品後の最終レイヤー

```
[製造・組立] → [検品・QC] → [本レイヤー: 法令適合確認] → [出荷]
```

本レイヤーは、品質検品を通過した製品が**市場投入・輸入・販売**の法的要件を満たしているかを確認する最終ゲートである。

---

## 1. United States (US)

### 1.1 電波・通信（FCC）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **Equipment Authorization** | 無線周波数機器は FCC 技術基準に適合。認証（Certification）または SDoC（Supplier's Declaration of Conformity）のいずれかで証明。 | Communications Act §302(b) |
| **FCC Part 15** | Wi-Fi、Bluetooth、Zigbee 等の免許不要機器。 | 短距離・ISM 帯 |
| **FCC Part 22 / 24 / 27** | LTE、5G、セルラー IoT 等の免許付与機器。 | TCB による認証必須 |
| **FCC ID** | 認証機器には FCC ID の表示。 | 本体またはラベル |
| **Covered Equipment** | 特定メーカーの通信・監視機器は認証・輸入・販売禁止。 | Secure Networks Act |

### 1.2 輸出管理（Export Control）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **EAR (Export Administration Regulations)** | BIS（商務省産業安全保障局）による品目分類。該当する場合は輸出ライセンスが必要。 | www.bis.doc.gov |
| **デュアルユース** | 民生品でも軍事転用可能な技術は規制対象。 | 品目・用途の確認 |

### 1.3 その他

| 項目 | 条件 | 備考 |
|-----|------|------|
| **EMC** | 妨害波・耐性の技術基準。FCC 認証に含まれる場合あり。 | |
| **SAR** | 人体近接型の無線機器は比吸収率（SAR）試験。 | 携帯・ウェアラブル等 |

## 2. Japan（日本）

### 2.1 電波法

| 項目 | 条件 | 備考 |
|-----|------|------|
| **技術基準適合証明** | 特定無線設備は登録証明機関または承認証明機関による適合証明。 | 総務省管轄 |
| **特定無線設備の例** | 特定小電力無線局、コードレス電話、携帯無線、MCA、構内無線、ラジオマイク等。 | 電波法施行規則 |
| **特性試験** | 占有周波数帯域幅、空中線電力、スプリアス発射強度等。 | 総務省告示第 88 号 |
| **技適マーク** | 適合証明を受けた機器には技適マークの表示。 | 本体またはラベル |
| **輸出専用** | 国内使用を前提としない輸出専用品は、輸出先国の規制に別途対応。 | |

### 2.2 電気用品安全法（PSE）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **事業届出** | 製造・輸入事業開始から 30 日以内に経済産業局へ届出。 | 保安ネット |
| **技術基準適合確認** | 全電気用品について自ら適合確認の義務。 | 法第 8 条第 1 項 |
| **特定電気用品** | 116 品目は登録検査機関の適合性検査、適合証明書の交付が必要。 | 457 品目中 |
| **自主検査** | 製造・輸入品について検査記録を 3 年間保存。特定電気用品は全数検査。 | |
| **輸出専用** | 専ら輸出する旨を届出に記載した場合、適合義務の免除あり。 | |

### 2.3 その他

| 項目 | 条件 | 備考 |
|-----|------|------|
| **EMC** | VCCI（情報技術機器の妨害波）等。無線機器は電波法に含まれる場合あり。 | |
| **RoHS** | 日本国内では法的義務はないが、EU 向け出荷時は EU RoHS 対応が必要。 | |

## 3. Germany / EU（ドイツ・欧州連合）

ドイツは EU 加盟国であり、EU 指令が国内法（FTEG 等）に転換されている。EU 域内への出荷は EU 指令が適用される。

### 3.1 電波機器指令（RED 2014/53/EU）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **CE マーキング** | 電波を送受信する機器は RED に基づく CE マークの表示が必須。 | ほぼ全ての無線機器 |
| **基本要件** | 安全、健康、EMC、電波スペクトラムの効率的利用。 | Annex III |
| **適合宣言（DoC）** | 技術文書、適合評価手順の実施。 | 製造者責任 |
| **輸入者責任** | 第三国から EU へ輸入する場合、輸入者が適合性を確認。 | |
| **言語** | 適合性に関する文書は対象国の言語で整備。 | |

### 3.2 EMC 指令（2014/30/EU）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **EMC** | 不要な電磁妨害を発しないこと、通常の妨害に対して耐性を持つこと。 | 電波機器は RED に含まれる場合あり |
| **調和規格** | 調和規格への適合で適合推定。 | |

### 3.3 RoHS 指令（2011/65/EU）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **有害物質制限** | 鉛、水銀、カドミウム、六価クロム、PBB、PBDE、特定フタル酸エステル等の使用制限。 | 電気電子機器 |
| **違反時** | 罰金、リコール、市場禁止の可能性。 | |

### 3.4 WEEE 指令（2012/19/EU）

| 項目 | 条件 | 備考 |
|-----|------|------|
| **廃電気電子機器** | 廃棄・リサイクルに関する要件。加盟国が国内法で実施。 | 製造者責任 |

### 3.5 ドイツ国内

| 項目 | 条件 | 備考 |
|-----|------|------|
| **Bundesnetzagentur** | 電波・通信機器の市場監視。FTEG（電気通信設備法）を所管。 | 独連邦ネットワーク庁 |
| **CE マーク** | EU 域内流通の前提。ドイツ国内でも有効。 | |

## 4. 出荷前チェックリスト（3 地域対応）

検品後、出荷前に以下を確認すること。

### 4.1 US 向け出荷

- [ ] FCC 認証（Part 15 / 22 / 24 / 27 等）の取得
- [ ] FCC ID の表示
- [ ] Covered Equipment に該当しないことの確認
- [ ] 輸出管理（EAR）の品目分類・ライセンス要否の確認
- [ ] SAR 試験（人体近接型の場合）

### 4.2 Japan 向け出荷

- [ ] 電波法：技術基準適合証明の取得（特定無線設備の場合）
- [ ] 技適マークの表示
- [ ] 電気用品安全法：事業届出、技術基準適合確認、自主検査記録
- [ ] PSE マーク（特定・特定外電気用品に応じて）

### 4.3 Germany / EU 向け出荷

- [ ] RED：CE マーク、適合宣言、技術文書
- [ ] EMC 指令：適合（RED に含まれる場合は RED でカバー可能）
- [ ] RoHS：有害物質の不使用確認
- [ ] WEEE：加盟国ごとの登録・報告（該当する場合）
- [ ] 対象国言語での取扱説明書・ラベル

## 5. プロジェクトでの設定

`00_Screening.md` またはプロジェクト設定で、**出荷対象地域**を明示すること。

| 項目 | 値 |
|-----|-----|
| **出荷対象地域** | US / Japan / Germany(EU) / その他（複数選択可） |
| **無線機能** | 有（LTE-M, Wi-Fi, Bluetooth 等） / 無 |
| **電気用品該当** | 有（特定 / 特定外） / 無 |
| **輸出管理要否** | 要確認 / 不要 |
